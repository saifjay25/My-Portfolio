Developed a portion of an app using Android Studio and ​Java​ named Peanuts that specifies ingredients in the food consumers may be allergic to. I developed the calorie counter of the app that adds calories, protein sugar, fiber, fats and carbohydrates of different foods. This app makes API calls to a MySQL database containing over 1000 foods with their nutrition and retrieves the data (RESTful web service) in JSON and displays the data that the user searched.

Programmed a Photo Album application in Java using Eclipse Oxygen. This app can have multiple users with a login for each user. When the user logs in they can view the albums in their account and the user can create, and delete albums. In the album they can view photos they had uploaded or add new ones and give it a caption or add location tags and name tags to the photo. Users can also search through photos by looking up a certain tag. When a photo is clicked, it shows when it was uploaded and enlarges the picture. There is also an administrator account that can delete or add new accounts to the app.
